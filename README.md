## 二、简介

​		P2note(Penetration-test payload note) 是基于layui 框架设计的一款渗透测试懒人工具，本意为**减少渗透测试人员记忆payload的数量，提升渗透测试效率**。

**更新日志** 2023-12-16

- 优化UI配色
- 新增LFI、MSF、Mimikatz、Webshell、反弹shell栏目
- 导航栏 IP_DN_Extract 新增fofa、hunter搜索引擎的语法生成功能
